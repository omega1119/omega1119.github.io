BeatBar Pocket (iOS)

Timing tools on the go.

BeatBar Pocket brings the same timing calculators to iPhone/iPad with a touch-first layout and a liquid bottom tab bar.

  • Tap Tempo, BPM → ms / Hz, Note ↔ Hz
  • Transpose / Speed helpers
  • FM calculator + LFO rates
  • Harmonic series explorer

BeatLab iOS (iOS)

Touch-first theory lab.

BeatLab iOS brings BeatLab’s detection, harmony generators, pivots/borrows, and interactive circles to iPhone/iPad in a touch-first layout.

  • Scale + Chord Finder (MIDI + touch keyboard)
  • Notes in Key, Scale Chords, Shared Notes/Chords
  • Mode pivots + Borrowed chords
  • Interactive Circle of Fifths + Circle of Thirds
  • Built for iPhone/iPad

macOS

BeatBar (macOS)

Desktop timing lab and calculators.

BeatBar is a windowed timing lab for fast, accurate conversions: tap tempo, BPM↔ms/Hz tables, note↔frequency lookups, transpose/speed helpers, FM + LFO calculators, and a harmonic series explorer.

  • Tap Tempo and shared BPM binding
  • BPM → ms / Hz table (straight/dotted/triplet)
  • Note ↔ Frequency table (tuning-aware)
  • Transpose / Speed (interval + BPM + ratio)
  • FM calculator, LFO rates, Harmonic series
Learn More

BeatLab (macOS)

Full theory workspace.

BeatLab is a desktop theory workspace for detecting chords/scales, generating diatonic harmony, exploring shared notes/chords, mode pivots and borrowed chords, plus interactive circles for modulation ideas.

  • Scale + Chord Finder (MIDI + on-screen keyboard)
  • Notes in Key, Scale Chords, Shared Notes/Chords
  • Mode pivots + Borrowed chord explorer
  • Circle of Fifths + Circle of Thirds
  • Interactive circles for modulation ideas
