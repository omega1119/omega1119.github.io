Features

BeatLab is a full theory workspace for detecting chords and scales, generating diatonic harmony, exploring shared notes/chords, and testing modulation ideas with interactive circles.

  • Scale + Chord Finder: live MIDI + on-screen keyboard intake with chord (or dyad) detection and ranked scale coverage
  • Notes in Key: spelled scale tones and an in-key keyboard highlight
  • Scale Chords: diatonic triads/sevenths per degree with inversion/extension controls and keyboard voicings
  • Shared Notes: compare two keys/scales and visualize overlap vs distinct tones
  • Shared Chords: compare diatonic chords between two keys/scales
  • Mode Pivots: find other modes/keys that contain the selected chord (with optional category filtering)
  • Borrowed Chords: borrowed options from parallel modes, grouped by source mode (triad/7th aware)
  • Circle of Fifths: interactive circle for exploring key relationships and modulations
  • Circle of Thirds: chromatic mediants for alternative modulation pathways
