BeatBar for macOS: Menu Bar Rhythm Toolkit

BeatBar is designed for musicians who value speed and precision. It sits in your menu bar, offering global tap tempo, timing and swing calculators, live chord and scale detection from MIDI, and transposition helpers.

Features

  • Global tap tempo and BPM calculator
  • MIDI chord/scale detection
  • Timing math and swing analysis
  • Transposition and note helpers
  • Minimal, distraction-free UI
  • Customizable global hotkeys
  • Built on modular Swift Packages (BeatCore, BeatInput, BeatUI, BeatFeatures, BeatMacKit)
