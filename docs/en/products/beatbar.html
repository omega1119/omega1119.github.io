Features

BeatBar is a desktop timing lab for fast, accurate conversions: tap tempo, BPM‚Üíms/Hz tables, note‚Üîfrequency lookups, transpose/speed helpers, FM + LFO calculators, and a harmonic series explorer.

  • Tap Tempo: instant and averaged BPM from taps, with optional shared BPM binding across tabs
  • BPM ‚Üí ms / Hz: straight/dotted/triplet values from whole to 16th notes, shown as ms + repetition rate
  • Note ‚Üî Frequency: per-octave table with MIDI number, Hz, period (ms), and samples at 44.1/48k; tuning is configurable
  • Transpose / Speed: key transpose + cents + speed slider expressed in semitones; outputs interval name, resulting BPM, and ratio
  • FM Calculator: ratio mode + pitch-correction mode, with sidebands and harmonicity hints
  • LFO Rates: tempo divisions to Hz/period/samples (48k), plus reverse lookup from custom Hz
  • Harmonic Series: first 32 harmonics with nearest note, cents offset, and interval labels

BeatBar is built with SwiftUI and CoreMIDI, with reusable logic shared across the BeatBar Suite via modular Swift packages.
