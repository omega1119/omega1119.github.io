BeatBar Pocket for iOS

A pocket timing toolkit with tap tempo and fast conversions for iPhone and iPad.

  • Tap Tempo + shared BPM binding
  • BPM → ms / Hz (straight/dotted/triplet)
  • Note ↔ Frequency (tuning-aware)
  • Transpose / Speed, FM + LFO, Harmonic series

Features

BeatBar Pocket shares the same calculator core as BeatBar, with a touch-first layout tuned for quick use during sessions, rehearsals, and travel.

  • Built for iPhone/iPad: touch-first layout with a liquid bottom tab bar, optimized for quick, one-handed checks
  • Tap Tempo: instant and averaged BPM from taps, with an option to bind a shared BPM across tools
  • BPM → ms / Hz Table: straight/dotted/triplet timing from whole notes to 16th notes, shown as milliseconds and repetition rate (Hz)
  • Note ↔ Frequency Table: per-octave lookup with MIDI number, Hz, period (ms), and samples at 44.1k/48k; tuning is configurable
  • Transpose / Speed: transpose and cents with a speed slider expressed in semitones; outputs interval name, resulting BPM, and ratio
  • FM Calculator: ratio mode plus pitch-correction mode, with sidebands and harmonicity hints
  • LFO Rates: tempo divisions to Hz/period/samples (48k), plus reverse lookup from a custom Hz value
  • Harmonic Series: first 32 harmonics with nearest note, cents offset, and interval labels
